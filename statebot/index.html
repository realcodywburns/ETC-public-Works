About Statebot


State bot provides a public service to reasearchers interestd in ethereum classic. Every 7 days, state bot exports the latest state in json format to the IPFS network and updates its smart contract.

Statebot is a truly decentralized application which utlizes Ethereum classic, IPFS, the MEW framework, and Javascript. It is entirely hosted on the Interplanitary file system and interates with its parent contract through public web3 api points. The parent contract is located at 0x28bfbf40bf22e3e78767fa3ba321e7311374ddda

If you are running an IPFS node please add statebot or the latest JSON to your node and help everyone by seeding for a few days.
